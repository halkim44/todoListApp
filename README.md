# Todo List App
an app that enable user to plan or manage their tasks easily and also create multiple groups of tasks for a specific goal.

## Tech/framework used
- [Bootstrap](https://getbootstrap.com/)

## Features
- create multiple Projects
- add todos
- set todos as complete
- checklist
